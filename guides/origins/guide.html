Batman Arkham Origins

Make the game more stable and unlock more frames.

Before beginning:
Buy the season pass on Steam. It is required for the community patch.

Overview

This is a short article explaining how to use some Arkham Origins configuration files and two community patches to improve the overall experience playing the game since it experiences consistent crashes and bugs and allows for modern day multiplayer.

Requirements

  • Windows 10 seems to be required for multiplayer, patching the game should work on any OS.
  • A text editor like Notepad++ is stongly recommended.
  • A Nexus Mods account.

Getting Started

Game settings are stored in the "BmEngine.ini" file located in the game's directory. This file can be edited with Notepad++ to uncap the frame rate. We'll also be improving the game's graphics in a minor way.

To start, navigate to the game's directory. This is usually located in "C:\Program Files (x86)\Steam\steamapps\common\Batman Arkham Origins\SinglePlayer\BMGame\Config".

Open the "BmEngine.ini" file with Notepad++ and search for "bSmoothFrameRate". Change the value to false. This will unlock the frame rate. Beyond that, as noted by this remaster mod, you can up the shadow resolution (MaxShadowResolution) to 4096 and use a larger texture pool size (TexturePoolSize) which allows you to change the value to whatever VRAM amount you have on your graphics card. You can also turn on PCSS shadows, if not already enabled. However, that would be more up to personal preference. Feel free to experiment with these config files, there are many mods which edit them for a variety of reasons.

Community Patch

The community patch is a must have for the game. It fixes a lot of bugs and crashes and in some cases, fixes the game crashing when using DirectX 11 graphics settings. To install, simply download the patch from Nexus Mods and simply run the program included in the downloaded archive.

Multiplayer

The game's multiplayer was shut down but offered a lot of fun. The community has made a patch to allow for modern day multiplayer. It simply requires you to run a server on your machine, then you can simply share with friends and play together. To start, download the patch here. Then, follow the video below.
